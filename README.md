# Farmers' Market Finder 

This Web App shows a map of Farmers' Markets around an area by entering a Zip Code.

## Getting Started

* [Click Me to Check It Out!](http://khamla719.github.io/FarmersMarketHack/)

## Tech Stack / Stuff We Used LOL

* [jQuery](https://jquery.com/)
* [Google Maps API](https://developers.google.com/maps/)
* [USDA National Farmers' Market API](https://search.ams.usda.gov/farmersmarkets/v1/svcdesc.html)
* [Snazzy Maps](https://snazzymaps.com/)
* [Heroku](https://www.heroku.com/)
